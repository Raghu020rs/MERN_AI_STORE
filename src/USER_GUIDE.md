# AI Tools Marketplace - User Guide

## How to Use the Tool Comparison Feature

The tool comparison feature allows you to compare up to 4 AI tools side-by-side to help you make informed decisions.

### Getting Started

#### Step 1: Add Tools to Comparison

There are multiple ways to add tools to your comparison list:

**Option A: From Tool Cards**
1. Find a tool you want to compare in any section (Home, Trending, Top Charts, etc.)
2. Hover over the tool card (on desktop) or tap it (on mobile)
3. Click the **three dots menu (⋮)** in the top-right corner
4. Select **"Add to Compare"** from the dropdown menu

**Option B: From Tool Detail View**
1. Click on any tool card to open its detailed view
2. Below the main action buttons, you'll see an **"Add to Compare"** button with a scale icon (⚖️)
3. Click this button to add the tool to your comparison

**Option C: From List View**
1. Switch to list view using the layout toggle (if available)
2. Click the **three dots menu (⋮)** on any tool
3. Select **"Add to Compare"**

#### Step 2: Manage Your Comparison List

Once you add tools, you'll see the **Comparison Bar** appear at the bottom of the screen:

- **View Selected Tools**: See all tools you've added (up to 4)
- **Remove Individual Tools**: Click the ✕ icon next to any tool name
- **Clear All**: Click "Clear All" to remove all tools at once
- **Tool Count**: Shows "Compare Tools (X/4)" - indicating how many tools you've selected

**Important Notes:**
- You can add a **maximum of 4 tools** for comparison
- If you try to add a 5th tool, you'll see an error message
- The comparison bar stays visible as you browse different pages
- Your selections are saved in your browser (persists across sessions)

#### Step 3: View the Comparison

When you have **at least 2 tools** selected:

1. Click the **"Compare Now"** button in the Comparison Bar
2. A comparison dialog will open showing all selected tools side-by-side

The comparison view includes:

- **Tool Headers**: Icons, names, and descriptions
- **Quick Actions**: Install/View buttons for each tool
- **Comparison Table**: Side-by-side comparison of:
  - Rating and reviews
  - Price (Free, Paid, Freemium)
  - Developer
  - Downloads
  - Category
  - Last Updated
  - Key Features (up to 5 per tool)

### Tips for Best Results

1. **Compare Similar Tools**: Compare tools in the same category for more meaningful insights
2. **Use Filters First**: Use the advanced filters to narrow down your options before comparing
3. **Check Multiple Aspects**: Look at ratings, features, and pricing together
4. **Mobile Friendly**: The comparison bar is positioned above the mobile navigation for easy access
5. **Persistent State**: Your comparison selections are saved, so you can browse and add more tools over time

### Keyboard Shortcuts & Accessibility

- The comparison feature is fully keyboard accessible
- Use Tab to navigate between tools in the comparison bar
- Press Escape to close the comparison dialog
- Screen reader announcements for all comparison actions

### Troubleshooting

**"Add to Compare" button is disabled?**
- You may have already added 4 tools (maximum limit)
- Remove at least one tool to add another

**Comparison bar not showing?**
- Make sure you've added at least one tool to comparison
- Check if you're scrolled to the top - the bar is fixed at the bottom

**Lost your comparison selections?**
- If you cleared your browser cache/cookies, selections will be reset
- Simply re-add your tools to start a new comparison
